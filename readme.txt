== Description ==
**Motive** Preloader with Quotes on website. Different pages can have different-2 quotes with beautiful templates. Multiple quotes can be added for one page and can also choose either random or in order. One Global Quote can be used for not chosen pages.

**Free Version**

1. 5 Quotes
2. Random or in order for pages
3. 1 Beautiful template

**Pro Version**

1. Unlimited Quotes
2. Random or in order for pages
3. Global Quote for not chose page
4. 5 Beautiful templates

http://vatsinformatics.com/preloader-quotes

**Support** After successful plugin Quizleads: https://codecanyon.net/item/quiz-leads/19935898?s_rank=1 highly energetic to get success for this too.

For a manual installation via FTP:

1. Extract the downloaded plugin and upload to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area
